Research

🔬 AI-Enhanced Identity Access Management

This research project investigates the application of machine learning algorithms to enhance Microsoft Entra ID's security capabilities. The study focuses on developing predictive models for anomalous user behavior detection and automated threat response systems in cloud-based identity management platforms.

Research Methodology & Findings:

  • Implemented machine learning models for behavioral analysis of user authentication patterns
  • Developed automated conditional access policy recommendations based on risk assessment algorithms
  • Created predictive models for identifying potential security breaches in real-time
  • Analyzed the effectiveness of AI-driven threat detection compared to traditional rule-based systems

Publications & Presentations:

  • Presented findings at UF Undergraduate Research Symposium 2024
  • Manuscript in preparation: "Machine Learning Applications in Cloud Identity Management"

🛡️ Honeypot Intelligence Systems

This research explores the integration of artificial intelligence with honeypot technologies to create more sophisticated deception systems. The project involves developing adaptive honeypots that can dynamically adjust their behavior based on attacker patterns using machine learning algorithms.

Research Contributions:

  • Designed intelligent honeypot systems using reinforcement learning algorithms
  • Analyzed attack patterns and developed predictive models for threat intelligence
  • Created automated response mechanisms for enhanced threat mitigation
  • Evaluated the effectiveness of AI-driven deception technologies
